# Adult-Census-Income-Dataset-Analysis



## About adult dataset : 

The Adult dataset, also known as the Census Income dataset, is a widely used dataset in machine learning. It was extracted from the 1994 Census database and is used to predict whether an individual's annual income exceeds $50,000 based on various demographic and employment-related features1.

##Key Features of the Adult Dataset:

- Number of Instances: 48,842

- Number of Features: 14

- Task: Binary classification (predicting whether income is above or below $50,000)

- Feature Types: Categorical, Integer

- Missing Values: Yes, marked with a question mark character (?)

##Features:

- age: Age in years

- workclass: Type of work (e.g., Private, Self-emp-not-inc, Federal-gov, etc.)

- fnlwgt: Final weight

- education: Education level (e.g., Bachelors, HS-grad, Masters, etc.)
  
-  educational-num: Number of years of education

- marital-status: Marital status (e.g., Married-civ-spouse, Divorced, Never-married, etc.)

- occupation: Occupation (e.g., Tech-support, Craft-repair, Exec-managerial, etc.)

- relationship: Relationship to householder (e.g., Wife, Own-child, Husband, etc.)

- race: Race (e.g., White, Asian-Pac-Islander, Amer-Indian-Eskimo, etc.)

- sex: Gender (Female, Male)

capital-gain: Capital gain amount

capital-loss: Capital loss amount

hours-per-week: Hours worked per week

native-country: Native country (e.g., United-States, Cambodia, England, etc.)
